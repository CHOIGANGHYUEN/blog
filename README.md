3. 강조
   이텔릭체는 _별표(asterisks)_ 혹은 *언더바(underscore)*를 사용하세요.

   두껍게는 **별표(asterisks)** 혹은 **언더바(underscore)**를 사용하세요.

   ***이텔릭체*와 두껍게**를 같이 사용할 수 있습니다.

   취소선은 ~~물결표시(tilde)~~를 사용하세요.

   <u>밑줄</u>은 `<u></u>`를 사용하세요.
